Proshop

Proshop is an e-commerce application developed for our Sofware Maintenance & Evolution assignment where users can buy electronic gadgets online using a secure and fast payment method using Paypal & Stripe. The evolution that we developed for this system is to create a parent payment in which a child can request to buy any item that will be paid by the parent.

Proshop was developed using React js & Redux for the frontend while for the backend, they are using Express js with MongoDB as their database. They also use services from Vercel to deploy the frontend and Heroku as a cloud hosting. Code is maintained at Github.
