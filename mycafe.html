Caféoo & Caféoo Merchant

Caféoo is a freelance project developed for consumers to explore restaurants in Malaysia registered in the app. It provides features like takeaway orders, dine-in orders, table reservations etc.

The application is built using React Native with Express Node js, Firebase and integrated with other external services such as map services, payment gateways etc. Code is maintained at Github. This mobile application has been successfully deployed to Google Play Store and Apple App Store undergoing testing phases.

The application is targeted to be released in production in 2022.

Contact me at zarifnazmi@gmail.com if you want to join the testing. Only available in Malaysia currently.

Caféoo Merchant

Caféoo Merchant is a freelance project developed for restaurant owners to handle orders, reservations and payments from Caféoo application users, employee management and many more.

The application is built using React Native with Express Node js, Firebase and integrated with other external services such as map services, payment gateways etc. Code is maintained at Github. This mobile application has been successfully deployed to Google Play Store and Apple App Store undergoing testing phases.

The application is targeted to be released in production in 2022.
